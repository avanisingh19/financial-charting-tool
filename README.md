# 📈 Market Data Visualizer

A web-based financial charting application for visualizing historical OHLC market data with candlestick charts and technical indicators.

## Features

- Upload OHLC CSV files
- Candlestick Chart
- Multiple Timeframes
- SMA
- EMA
- DEMA
- WMA
- RSI
- MACD
- ATR
- Bollinger Bands
- CCI
- ROC
- Momentum
- Williams %R
- DMI

## Technologies

- HTML5
- CSS3
- JavaScript
- Highcharts Stock
